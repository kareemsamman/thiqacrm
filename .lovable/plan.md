## السبب الجذري

نافذة `DebtPaymentModal` تحسب كل وثيقة بشكل مستقل:
```tsx
remaining: p.insurance_price - (policyPayments[p.id] || 0)
```

لكن ملف العميل يحسب إجمالي السعر - إجمالي المدفوع = المتبقي الفعلي

**النتيجة:** الزيادة في دفع الإلزامي (1,000₪) تُخصم من المتبقي في ملف العميل، لكن لا تظهر في نافذة الدين.

---

## الحل المقترح

### استخدام نفس منطق حساب ملف العميل

بدلاً من حساب كل وثيقة بشكل مستقل، نحسب:

1. **إجمالي أسعار الوكالة** (غير الإلزامي)
2. **إجمالي المدفوع على كل الوثائق** (بما فيها الإلزامي)
3. **إجمالي أسعار الإلزامي**
4. **صافي دين الوكالة** = أسعار الوكالة - (إجمالي المدفوع - أسعار الإلزامي)

### الصيغة الجديدة:
```
صافي الدين = أسعار الوكالة - الفائض من دفعات الإلزامي
```

في حالة Kareem Test:
```
أسعار الوكالة = 4,530₪
إجمالي المدفوع = 8,055₪
أسعار الإلزامي = 6,555₪
الفائض = 8,055 - 6,555 = 1,500₪ (مدفوع لوثائق الوكالة)
- منها 500₪ للطريق (باقة 1)
صافي الدين = 4,530 - 1,500 = 3,030₪ ✅
```

## التغييرات التقنية

### الملف: `src/components/debt/DebtPaymentModal.tsx`

#### 1. إضافة حساب الدفعات الزائدة

```tsx
// حساب إجمالي جميع الدفعات (بما فيها الإلزامي)
const { data: allPaymentsData } = await supabase
  .from('policy_payments')
  .select('amount, refused, policy:policies!inner(client_id)')
  .eq('policy.client_id', clientId)
  .eq('refused', false);

const totalAllPayments = (allPaymentsData || []).reduce((sum, p) => sum + p.amount, 0);

// حساب إجمالي أسعار كل الوثائق
const { data: allPolicies } = await supabase
  .from('policies')
  .select('id, insurance_price, policy_type_parent')
  .eq('client_id', clientId)
  .eq('cancelled', false)
  .is('deleted_at', null);

const totalAllPrices = (allPolicies || []).reduce((sum, p) => sum + p.insurance_price, 0);

// صافي الدين = إجمالي الأسعار - إجمالي المدفوع
const netDebt = Math.max(0, totalAllPrices - totalAllPayments);
```

#### 2. تصفية البطاقات المعروضة

عرض فقط البطاقات التي فيها رصيد متبقي **حقيقي** بعد احتساب الفوائض.

#### 3. إظهار الإحصائيات الصحيحة

```tsx
<div className="bg-muted/50 rounded-lg p-3 text-center">
  <p className="text-xs text-muted-foreground">إجمالي الدين</p>
  <p className="text-lg font-bold ltr-nums">₪{netDebt.toLocaleString('en-US')}</p>
</div>
```
