
# تحسين قسم "النشاط الأخير" - نسخة احترافية كاملة

## المشاكل الحالية

بناءً على مراجعة الكود والصورة المرفقة:

1. **لا يمكن رؤية جميع النشاطات** - يعرض فقط آخر 5 عناصر
2. **لا يوجد فلترة بالتاريخ** - من تاريخ إلى تاريخ
3. **لا يوجد بحث** - للعثور على نشاط محدد
4. **نوع الدفعة مفقود** - لا يظهر (كاش/شيك/فيزا/حوالة)
5. **المبلغ غير مجموع** - للدفعات المتعددة
6. **تفاصيل الوثيقة ناقصة** - أي وثيقة، من أي شركة
7. **المعلومات سطحية** - تحتاج تفاصيل أكثر احترافية

---

## الحل المقترح

### 1. إنشاء صفحة كاملة للنشاط `/activity`

بدلاً من الاكتفاء بالبطاقة الصغيرة على Dashboard، سيتم إنشاء صفحة كاملة تحتوي على:

| الميزة | الوصف |
|--------|--------|
| **فلترة بالتاريخ** | من تاريخ → إلى تاريخ مع Date Picker |
| **بحث نصي** | البحث في اسم العميل، رقم السيارة، أي شيء |
| **فلترة بالنوع** | وثيقة / دفعة / عميل / سيارة |
| **عرض أكثر** | صفحات Pagination أو Load More |
| **تفاصيل كاملة** | كل المعلومات المطلوبة |

### 2. بطاقة Dashboard محسّنة

البطاقة على الـ Dashboard ستبقى ولكن مع تحسينات + رابط "عرض الكل" للصفحة الكاملة.

### الميزات الرئيسية

1. **فلتر التاريخ**:
   - `من تاريخ` + `إلى تاريخ` باستخدام Date Picker
   - اختصارات سريعة: "اليوم"، "هذا الأسبوع"، "هذا الشهر"

2. **البحث**:
   - اسم العميل
   - رقم الملف
   - رقم السيارة
   - اسم الموظف

3. **فلتر النوع**:
   - الكل
   - دفعات فقط
   - وثائق فقط
   - عملاء جدد
   - سيارات

4. **ملخص مالي** (للدفعات):
   - عند تصفية الدفعات يظهر: `مجموع الدفعات: ₪45,000`

5. **تحميل المزيد / Pagination**:
   - يعرض 20 عنصر في البداية
   - زر "تحميل المزيد" أو أرقام صفحات
